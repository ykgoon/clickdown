## REMOVED Requirements

### Requirement: Inbox navigation entry point
**Reason**: The global "Inbox" sidebar item is being removed. Cross-workspace activity aggregation is replaced by per-list filtering.

**Migration**: No direct replacement. Users navigate to individual lists to view tasks and comments relevant to that context.

### Requirement: Inbox navigation state
**Reason**: The dedicated inbox screen and its navigation state are removed along with the Inbox sidebar item.

**Migration**: No replacement needed. The application no longer has a global inbox view.
