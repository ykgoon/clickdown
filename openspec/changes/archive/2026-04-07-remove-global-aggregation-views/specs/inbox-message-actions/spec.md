## REMOVED Requirements

### Requirement: Clear notification action
**Reason**: The inbox notification clear action is removed along with the entire inbox feature.

**Migration**: No direct replacement. Users manage their work through per-list filtering rather than a centralized notification queue.

### Requirement: Clear all notifications action
**Reason**: Bulk notification clearing was specific to the inbox feature.

**Migration**: No replacement needed.

### Requirement: Keyboard shortcut help
**Reason**: Inbox-specific keyboard shortcuts (c, C for clearing notifications) are removed along with the inbox feature.

**Migration**: The help dialog no longer displays inbox shortcuts. List-view shortcuts remain unchanged.

### Requirement: Notification detail view
**Reason**: The notification detail panel was part of the inbox UI, which is being removed.

**Migration**: No replacement. Task detail view remains accessible from the standard list view.
